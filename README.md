# BoringXMLEditor

ボーリングXMLをタブ画面で編集できるWindowsアプリです。
DTD 2.10 / 3.00 / 4.00 に対応しています。

## ダウンロード

[最新版ダウンロード](https://github.com/mipcivil/BoringXMLTool-JP/releases/latest)

インストール不要。ZIPを展開して `BoringXMLEditor_*.exe` を実行するだけで使えます。

## Windows SmartScreen の警告について

初回起動時に「WindowsによってPCが保護されました」と表示される場合があります。
これは個人開発のアプリにコード署名証明書が付与されていないために表示されるもので、ソフトウェア自体に問題はありません。

1. 「詳細情報」をクリック
2. 「実行」ボタンをクリック

この操作は初回のみ必要です。
