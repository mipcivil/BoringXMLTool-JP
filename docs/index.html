主な機能

複数DTDバージョン対応

DTD 2.10、3.00、4.00 を自動検出し、適切なフォーマットで処理します。

直感的なUI

タブ形式のインターフェースで、調査情報・土層情報・基本情報を効率的に編集できます。

XML読み込み・保存

日本語XMLデータの読み込み、編集、保存をシームレスに行えます。

プレビュー・PDF出力

ボーリング柱状図のプレビューとPDF・DXF形式でのエクスポートに対応しています。

サービス概要

BoringXMLEditor（以下「本ソフトウェア」）は、地質調査ボーリングXMLデータの編集・管理を目的としたデスクトップアプリケーションです。月額または年額のサブスクリプション方式でライセンスを提供します。

ライセンス

  • ライセンスは購入者のマシン1台に紐づきます。
  • ライセンスキーの第三者への譲渡・共有は禁止します。
  • サブスクリプションは選択したプランに応じて月単位または年単位で自動更新されます。

支払い

  • 決済は Stripe を通じて安全に処理されます。

  • 価格は予告なく変更される場合があります。既存のサブスクリプションには次回更新時から適用されます。
